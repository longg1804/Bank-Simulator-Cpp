**How To Use:**
- Open _bank.cpp_ file by any C++ compiler and run this file to start using Bank Simulator.
- Account list will be auto-saved at _accountdata.txt_ file as a format _login_name|password|name|account_name|balance_.

**Note:**
- The program is currently available **only on Windows**. I will be releasing support for other platforms soon.
